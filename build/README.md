# Perspectiva WordPress Theme

Тема WordPress для сайта Perspectiva - производство фиброцементных панелей и фасадных систем.

## Установка

1. Скопируйте папку `build` в директорию `wp-content/themes/` вашего WordPress сайта
2. Переименуйте папку в `perspectiva` (или оставьте как есть)
3. Войдите в админ-панель WordPress
4. Перейдите в `Внешний вид` → `Темы`
5. Активируйте тему "Perspectiva"

## Настройка

### Меню

1. Перейдите в `Внешний вид` → `Меню`
2. Создайте новое меню или используйте существующее
3. Добавьте пункты меню
4. В разделе "Расположение меню" выберите "Основное меню"
5. Сохраните изменения

### Виджеты

1. Перейдите в `Внешний вид` → `Виджеты`
2. Добавьте виджеты в область "Сайдбар"

## Структура темы

```
build/
├── style.css          # Заголовок темы (обязательный файл)
├── functions.php      # Функции темы
├── index.php          # Главный шаблон
├── header.php         # Шапка сайта
├── footer.php         # Подвал сайта
├── page.php           # Шаблон страниц
├── single.php         # Шаблон записей
├── archive.php        # Шаблон архивов
├── 404.php            # Шаблон 404 ошибки
├── assets/            # Ресурсы (CSS, JS, изображения)
├── components/        # Компоненты темы
│   ├── elements/      # Элементы
│   └── sections/      # Секции
└── inc/               # Вспомогательные файлы
```

## Особенности

- Поддержка миниатюр записей
- Поддержка кастомных логотипов
- HTML5 разметка
- Адаптивный дизайн
- Автоматическое версионирование CSS и JS файлов

## Требования

- WordPress 5.0 или выше
- PHP 7.4 или выше
